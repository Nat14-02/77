Visual Studio es una app o una plataforma en la cual puedes generar paginas web con distintos lenguajes de progrmacion y diferentes metodos de este mismo.

Para configurarlo debes:

Asegurarse de que el equipo está listo para Visual Studio

Determinar qué versión y edición de Visual Studio se van a instalar

Iniciar la instalación

Elegir las cargas de trabajo

Elegir componentes individuales (opcional)

Instalar paquetes de idioma (opcional)

Seleccionar la ubicación de instalación (opcional)

Empezar a desarrollar
